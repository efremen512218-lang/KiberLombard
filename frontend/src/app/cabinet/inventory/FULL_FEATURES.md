# Полный функционал инвентаря

## Что нужно добавить:

### 1. Загрузка инвентаря по ссылке
- Кнопка "Загрузить другой инвентарь"
- Поле ввода ссылки на Steam профиль
- Парсинг Steam ID из ссылки
- Загрузка инвентаря

### 2. Функционал кнопок
- "Выбрать все" - выбрать все отфильтрованные скины
- "Сбросить" - снять выделение
- "Продолжить" - перейти к созданию сделки
- Поиск - фильтрация в реальном времени
- Сортировка - по цене/названию
- Фильтр редкости - показать только выбранную редкость

### 3. Реальный Steam API
Для загрузки реального инвентаря нужен Steam API ключ.

Форматы ссылок:
- https://steamcommunity.com/profiles/76561198000000000/inventory
- https://steamcommunity.com/id/username/inventory
- 76561198000000000 (просто ID)

1. Получить ключ: https://steamcommunity.com/dev/apikey
2. Добавить в .env: STEAM_API_KEY=xxx
3. Backend будет загружать реальный инвентарь
