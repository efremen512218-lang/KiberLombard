# ⚡ Обновлено: Модель 50% от маркетплейсов

## Что изменилось

**Было:** Сложная градация коэффициентов (50-65% в зависимости от цены)

**Стало:** Простая модель - всегда 50% от instant sell цены

## Новая формула

```
Залог = min(Lis-Skins instant, market.csgo instant) × 50%
```

### Пример:
- AWP | Asiimov на Lis-Skins: 8 000₽
- AWP | Asiimov на market.csgo: 8 200₽
- **Минимум:** 8 000₽
- **Залог:** 8 000 × 50% = **4 000₽**

## Преимущества

### 1. Прозрачность
Клиент видит цену на маркетплейсах и понимает откуда берётся оценка.

### 2. Простота
Одна формула для всех предметов. Легко объяснить.

### 3. Честность
Используем реальные рыночные цены, не свои оценки.

### 4. Конкурентность
50% - это больше чем у других ломбардов (обычно 30-40%).

### 5. Безопасность
При дефолте продаём за 100% → гарантированная прибыль 100%.

## Изменения в коде

### Backend: `pricing_service.py`
```python
@staticmethod
def get_purchase_coefficient(instant_price: float) -> float:
    return 0.50  # Всегда 50%
```

### Frontend: `inventory/page.tsx`
```typescript
const loanAmount = Math.round(instantPrice * 0.50)  // Всегда 50%
```
